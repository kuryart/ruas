# Exibição do nome das notas

- O nome dos arquivos estão sendo salvos como "arquivo (1)" quando já existe um arquivo de mesmo nome. Porém, isso não está sendo refletido no nome das notas/contatos. Quando se cria uma nota/contato com mesmo nome, o frontend deve seguir a mesma lógica utilizada para o nome dos arquivos. Exemplo "nota (1)" quando já existe uma nota chamada "nota", "contato (1)" quando já existe um contato chamado "contato"
